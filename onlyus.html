🔒 Security breakdown

Passphrase → 256-bit key
PBKDF2-SHA512 600,000 iterations (OWASP 2023 standard). Output is always exactly 32 bytes (256 bits). Even `1432` produces a 256-bit key that cannot be reversed.

Double-lock key derivation
X25519 ECDH + PBKDF2 hash
Final key = SHA-512(X25519_result ‖ PBKDF2_result) → first 256 bits.
Both must be correct to decrypt anything.

Every message & photo
XSalsa20-Poly1305 Same as Signal. 192-bit random nonce per message. Authenticated — tamper-proof.

Video & voice
DTLS-SRTP WebRTC browser-enforced E2EE. Relay sees zero media.

No storage
Nothing saved anywhere. Close tab = everything gone forever.
