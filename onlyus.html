🔒 Full security breakdown

Passphrase → 256-bit key
Argon2id — winner of Password Hashing Competition 2015. Memory-hard (64MB RAM used during hashing). Time-hard (3 iterations). Even with a GPU farm cracking `1432` would take longer than cracking AES-256 by brute force. Output is always exactly 256 bits (32 bytes).

Key exchange
X25519 ECDH — elliptic curve Diffie-Hellman. Your private key never leaves your device. The shared secret is mixed with the Argon2 key — both must be correct to decrypt anything.

Every message & photo
XSalsa20-Poly1305 — authenticated encryption. Same algorithm as Signal. Unique 192-bit random nonce per message. Tamper detection built in.

Video & voice
DTLS-SRTP — WebRTC browser-enforced E2EE. The relay sees two anonymous IDs — zero media.

Key derivation chain
Argon2id(pass) → 256-bit hash
X25519(mySecret, theirPublic) → DH
SHA-512(DH ‖ Argon2Hash) → 512-bit
→ take first 256 bits → final SK

Storage
Zero. No database, no cookies, no logs. Close tab = gone forever.
